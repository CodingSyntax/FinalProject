# Journeyman's Quest
Journeyman's Quest is a text-based RPG made in Java. It has the following features:

* Fully fleged character
  * Levels and xp
  * Scoring
* Encounters
  * Cities
  * Enemy battles
  * Wandering trader
* Map
* Saving and loading
* And more!
